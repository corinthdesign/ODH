Menu Update System

Input the new menuboard item into the appropriate field. Fields left alone will not update on the menuboard. To finish updating the menu, press "Update Menu". Use the menu preview to ensure that the menu was updated properly. (*Note: Menu cannot handle " ' " characters. The menu will return a "null" item.).
